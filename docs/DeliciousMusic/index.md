# 好吃的唱片店：
通过DC服务器的自定义唱片功能以及交达音乐平台制作唱片，主打一个发行签约歌手的唱片公司，当然也会发行一些现有音乐打包成专辑发行。

本专栏主要记录已发行或代发行的专辑。
