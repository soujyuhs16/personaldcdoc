
# memon_tree的个人DC相关小文档

本网站主要是收集了memon_tree编写的有关DC服务器的文档，包括个人的DC教程，及其我主导的唱片店的相关文档

## 关于个人DC文档
本着[DC商业梦想城服务器](https://mcrail.top)的社区建设精神，本人向部分玩家收集了一些对DC服务器最常遇到的问题，或者是个人对于DC服务器文档的一些没有的描述的、描述得可能比较难以理解的内容，以及对自己长时间在DC服务器游玩的经验总结，所编写的一套属于自己的个人DC小文档。

关于最详细的服内游玩介绍，建议访问官方的[DC服务器文档](https://doc.mcrail.top)。本文档仅供参考补充使用

  
文档的内容仅是个人见解，如仍有不理解的地方，可以咨询本人memon_tree或服内其他玩家，或者通过官网提交工单来获得管理员的帮助。

## 关于好吃的唱片店
