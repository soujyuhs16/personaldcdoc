# 信任权限组的获取

根据DC服务器文档关于[[玩家权限组 - DC商业梦想城文档](https://doc.mcrail.top/help/sur/permission/player/)\]的介绍，获取信任权限组可以获得更多的home传送点和领地数量，同时可以进行自助地图画的申请以及CoreProtect插件的使用权。获取信任权限组需要你在游戏内在线时长达到10天，DC币余额超过8000dcb，不处于权限失信，无生效封禁等。  
关于信任权限组的获取这里简单给出图文教程。

查询条件是否满足

输入指令/playtime，会显示出你的游玩时间，确定游戏内在线时长达到10天。

输入指令/money，查询dcb余额是否超过8000，如不够，可以进行收购获得dcb，或者向别的玩家借款。

其实确定游玩时间和dcb余额是否满足要求后，即可尝试进行获取信任权限组的操作了。如果不放心的话，继续确定是否处于权限失信：输入指令/12306，检查弹出的菜单界面是否有屏障图标，如果没有则不处于权限失信，如果有，点一下屏障图标“手动结算”即可恢复。（其实就是你坐火车忘给钱了，但是一般都是自动扣费，很少出现权限失信情况。）关于生效封禁，都被封禁了连服务器肯定都进不去好吧。

## 获取信任权限组

（本办法已过时，请前往DC邮局任一网点，在DC自助服务机操作，详细操作流程请查阅[DC自助服务机 - DC商业梦想城文档](https://doc.mcrail.top/help/sur/servicemachine/trustedgrant/))
