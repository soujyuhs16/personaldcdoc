# 使用安卓手机游玩教程
相关文件请联系az_bai_cat或本人获取，亦作为如何配置FCL和MobileGlues渲染器的教程使用。

## 安装FCL

如果是从qq下载请点击右上角三个点，然后选保存到手机

![image-20260220235241152](https://b2.313511.xyz/image-20260220235241152_1771604346998_p46172.png)![image-20260220235248109](https://b2.313511.xyz/image-20260220235248109_1771604364628_pnk0vw.png)

然后在文件管理器找到以下路径，找到压缩包并解压到一个好找的目录（如果不是qq下载在下载文件夹里找一找）

 ![image-20260220235258142](https://b2.313511.xyz/image-20260220235258142_1771604378778_see81b.png)![image-20260220235304939](https://b2.313511.xyz/image-20260220235304939_1771604387168_g2d8eh.png)

找到这个目录，先安装这个目录下的两个应用（MobileGlues和FCL）

![image-20260220235312626](https://b2.313511.xyz/image-20260220235312626_1771604397588_gggtlu.png)

 

打开FCL（附魔台图标的应用（上面图二中左边标注的那个应用））

![image-20260220235319873](https://b2.313511.xyz/image-20260220235319873_1771604415107_rfxlyg.png)

 

点击确认

给权限打开

![image-20260220235326247](https://b2.313511.xyz/image-20260220235326247_1771604421928_69xsmc.png)

点同意并继续

![image-20260220235351801](https://b2.313511.xyz/image-20260220235351801_1771604434217_p0sgzj.png)

点安装/更新

![image-20260220235401651](https://b2.313511.xyz/image-20260220235401651_1771604442778_7ttpkh.png)

点左侧边栏一个云的图标

![image-20260220235410784](https://b2.313511.xyz/image-20260220235410784_1771604452217_t4qw2y.png)

点“整合包”

![image-20260220235418533](https://b2.313511.xyz/image-20260220235418533_1771604466508_2yet5o.png)

点安装整合包

![image-20260220235428296](https://b2.313511.xyz/image-20260220235428296_1771604766218_39k6hb.png)

导入本地整合包（memon_tree提供有在线链接，也可以询问其以使用从互联网下载整合包）

![image-20260220235438056](https://b2.313511.xyz/image-20260220235438056_1771604473479_4wsy4a.png)

找到你解压压缩包的目录，选择.mrpack结尾的文件并确认

![image-20260220235450995](https://b2.313511.xyz/image-20260220235450995_1771604516748_xgd9j8.png)

 

然后点击安装即可

![image-20260220235501702](https://b2.313511.xyz/image-20260220235501702_1771604525797_hwfafk.png)



## MobileGlues渲染器设置

![image-20260220235525821](https://b2.313511.xyz/image-20260220235525821_1771604539387_yc9375.png)

点击修改渲染器设置

![image-20260220235914708](https://b2.313511.xyz/image-20260220235914708_1771604549150_czhmwf.png)

点击确认

![image-20260220235925548](https://b2.313511.xyz/image-20260220235925548_1771604551899_2y76zr.png)

在根目录创建一个名字叫MG的文件夹

![image-20260220235945445](https://b2.313511.xyz/image-20260220235945445_1771604563887_j0t4yf.png)

点击MG然后选择使用此文件夹，弹出的窗口选择允许

![image-20260221000006790](https://b2.313511.xyz/image-20260221000006790_1771604567933_cspfd7.png)

出现以下界面说明MG渲染器安装成功

![image-20260221000015113](https://b2.313511.xyz/image-20260221000015113_1771604575767_jyt5kn.png)

回到FCL，点击左侧边栏的扳手图标

![image-20260221000021548](https://b2.313511.xyz/image-20260221000021548_1771604578037_tqcceq.png)

点击MobileGlues渲染器

![image-20260221000028678](https://b2.313511.xyz/image-20260221000028678_1771604579867_l6t6al.png)

然后创建用户启动游戏即可
