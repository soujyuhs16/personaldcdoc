# 梦幻聚落介绍
![梦幻](https://b2.313511.xyz/mw.png)  
梦幻（Mungwaan）成立于2024年8月16日，又称梦州或桥城，是DC服务器东北部的新兴城市。梦幻中部多山脉，两侧地势平坦，环境优美，非常适合居民居住与置业。梦幻的市政中心位于永昌区。光武区作为本市的经济中心，拥有众多商业建筑，经济发达。平海区则是梦幻铁路与公路的交通枢纽，同时也是最初的居民区。四通八达的道路网络能够便捷地通往市内各个区域，同时亦有方便的铁路可以通往DC各地。随着边界的扩张，彼方区、养马堡县（这俩为新居民区）、松山区（景区）和松南区（暂未开发）也在快速发展当中。欢迎各位玩家前来梦幻参观的同时也欢迎各位玩家参与梦幻的建设！
