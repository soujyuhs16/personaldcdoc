# 服务器常见问题（必读）

## 卡交互问题

就是如果你有时候打开不了箱子开不了门，聊天栏又没有任何报错提示，那大概率是卡下蹲了，按一下shift下蹲一下就恢复了。目前这个问题暂时无解

## 被限制乘车，处于权限失信

如图，有神秘新人风行尝试坐车但是它坐不了

![image-20260225213503186](https://b2.313511.xyz/image-20260225213503186_1772026762830_7cls4v.png)

原因：坐火车因为各种原因自动扣费失败，需要手动去给钱

如何处理？很简单，输入指令`/12306`，点一下屏障，即可消除权限失信。

![image-20260225213739993](https://b2.313511.xyz/image-20260225213739993_1772026794266_ep64ws.png)
